Projeto - Notas de Troca 

Um problema que eu e meus parentes sempre tivemos foi com notas de troca física, sempre eram várias notas, de lojas diferentes, com probabilidades de perda. Então, a ideia é um aplicativo mobile que conterá suas notas de troca, ao invés de uma nota fisica. O usuário poderá acessar suas notas de troca no próprio aplicativo. Cada loja, seja de roupas, eletrônicos ou qualquer outra, poderá registrar a nota de troca para o usuário.
